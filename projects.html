Assembly Line Layout Optimization & Process Improvement

Organization: Tritorc Manufacturing
Duration: 6 months
Role: Manufacturing Engineer Intern

Project Overview

Led comprehensive analysis and redesign of assembly line operations for automotive component manufacturing. The project focused on improving production efficiency, reducing waste, and optimizing material flow through detailed time studies, layout modifications, and lean manufacturing implementation.

Problem Statement

The existing assembly line suffered from inefficient material flow, excessive operator movement, and bottlenecks that limited production capacity. Cycle time variability was high, and workstation layouts did not follow ergonomic best practices, leading to operator fatigue.

Methodology & Tools

  • Time Studies: Conducted detailed time and motion studies across 12 workstations to establish baseline cycle times and identify non-value-added activities
  • Value Stream Mapping: Mapped current state to visualize waste and opportunities for improvement
  • Spaghetti Diagrams: Tracked operator movement patterns to quantify unnecessary travel distance
  • CAD Layout Design: Used SolidWorks to design optimized workstation layouts with improved tool placement and material flow
  • Standard Work Documentation: Created standardized work instructions with visual aids to reduce cycle time variation

  • Installed gravity-fed component racks to minimize reaching and bending
  • Trained 15 operators on new standard work procedures

Ergonomic Workstation Design & Human Factors Analysis

Organization: Clemson University Research
Duration: Ongoing
Role: Graduate Research Assistant

Project Overview

Research-focused project investigating ergonomic design principles for manufacturing workstations. Applied human factors engineering methods to assess and improve workstation design, reduce musculoskeletal strain, and optimize operator comfort and productivity.

üßç Ergonomic Analysis Placeholder

Research Methodology

  • RULA/REBA Analysis: Conducted rapid ergonomic assessments to identify high-risk postures and movements
  • Anthropometric Data Analysis: Applied percentile-based design principles to accommodate operator diversity
  • 3D Motion Capture: Analyzed operator movements and reach envelopes using motion tracking technology
  • Ergonomic Simulation: Used Jack/Siemens Human simulation software to validate workstation designs
  • Fatigue Assessment: Measured operator fatigue through subjective ratings and physiological monitoring

Key Findings

  • Identified 8 workstations with RULA scores above 5 (requiring immediate intervention)
  • Documented excessive forward bending (>30¬∞) and overhead reaching in 40% of tasks
  • Quantified force requirements exceeding recommended limits for repetitive pushing/pulling operations
  • Developed design guidelines for adjustable-height work surfaces and tool presentation systems

Design Recommendations

  • Adjustable work surface heights (28-44 inches) to accommodate 5th-95th percentile operators
  • Tool balancers and articulating arms to eliminate sustained arm elevation
  • Angled component presentation to reduce forward bending and improve visibility
  • Anti-fatigue matting and sit-stand capability for long-duration tasks
  • Task rotation schedules to distribute physical demands across operators

Cross-Dock Material Flow Optimization

Organization: DHL Global Forwarding
Duration: 4 months
Role: Operations Trainee

Project Overview

Redesigned cross-dock facility layout to improve material flow efficiency, reduce handling time, and minimize congestion. Project involved facility layout analysis, simulation modeling, and implementation of lean logistics principles.

Problem Analysis

The existing cross-dock layout resulted in excessive pallet movement, frequent forklift traffic conflicts, and inefficient dock door utilization. Average dwell time was 4.2 hours, significantly above industry benchmarks.

  • Analyzed 3 months of shipment data covering 15,000+ pallet movements
  • Created flow matrix to identify high-frequency origin-destination pairs
  • Mapped current material flow patterns and identified crossing paths
  • Documented dock door utilization rates and staging area bottlenecks

Solution Implementation

  • Reassigned dock doors based on frequency analysis to minimize travel distance
  • Created dedicated staging zones for high-velocity SKUs near destination doors
  • Implemented directional flow patterns to eliminate crossing forklift traffic
  • Established visual management system with floor markings and zone identification
  • Optimized pallet storage positions using ABC analysis

Poka-Yoke Implementation & Quality System Development

Organization: Tritorc Manufacturing
Duration: 3 months
Role: Manufacturing Engineer Intern

Project Overview

Designed and implemented error-proofing devices (Poka-Yoke) to eliminate defects at the source and reduce quality escapes. Project included root cause analysis, fixture design using CAD, and validation through capability studies.

üîß Poka-Yoke Device Placeholder
CAD models of error-proofing fixtures will be added here

Problem Identification

Analyzed 6 months of quality data revealing recurring defects in assembly operations:

  • Component misalignment causing 23% of defects
  • Incorrect part orientation accounting for 18% of rework
  • Missing fasteners identified in 15% of quality audits
  • Torque specification errors in critical joints

Poka-Yoke Solutions Implemented

  • Alignment Fixtures: Designed CAD models and fabricated fixtures with alignment pins to ensure correct component positioning
  • Asymmetric Parts Trays: Created custom trays that only accept parts in correct orientation
  • Count-Down Bins: Implemented visual counting system with pre-staged kits to prevent missing components
  • Torque-Sensing Tools: Integrated torque wrenches with data logging and visual/audible alerts
  • Go/No-Go Gauges: Designed and implemented inspection gauges at critical control points
